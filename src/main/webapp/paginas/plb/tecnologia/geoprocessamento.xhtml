banco de dados geográficos

Bancos de Dados Geográficos: São banco de dados preparados para armazenar dados sobre informações espaciais. São comumente chamados de Sistemas de Informações Geográficas. Este tipo de banco de dados tem o intuito manipular um imenso volumes de informações de grande complexidade, como mapas e imagens de satélite.
