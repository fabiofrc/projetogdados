TECNOLOGIA UTILIZADA NO PROJETO

A GDADOS é um projeto relacionado a ciências de dados, com foco nas seguinte tecnologias:

I - Criação e desenvolvimento de estrutura para Big Data;

II - Aplicação de algoritmo de aprendizado de máquina;

III - Análise de dados;

IV - Desenvolvimento de Dashboard Inteligente;

V - Integração de banco de dados;



Todas essas tecnologias envolvida e aplicada de forma eficiênte, gera valores e soluções para tomada decisão eficiente;
